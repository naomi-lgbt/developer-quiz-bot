# Developer Quiz Bot

This is just a bot that allows you to take freeCodeCamp's [Developer Quiz](https://developerquiz.org/) directly on Discord.

## Live Version

This app is currently deployed. [View it in the freeCodeCamp Discord.](https://discord.gg/KVUmVXA)
